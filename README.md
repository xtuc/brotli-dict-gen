# Brotli Dictionary Generator

Install:
```
cargo install brotli-dict-gen
```

Usage:
```
Brotli Dictionary Generator

Usage: brotli-dict-gen --output <OUTPUT> [FILES]...

Arguments:
  [FILES]...

Options:
  -o, --output <OUTPUT>  Path of the generated dictionary
  -h, --help             Print help
  -V, --version          Print version

```
